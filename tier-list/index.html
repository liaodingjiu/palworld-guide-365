Palworld Tier List

S/A/B rankings for Combat, Base Work, and Overall. All 215 Pals ranked by verified stats from palworld.wiki.gg. Methodology: Combat = Attack stat. Worker = total work suitability. Overall = combined. S = top 10%, A = next 20%, B = next 30%.

📊 Methodology: All rankings use verified data from palworld.wiki.gg. Combat tier = Attack stat (raw damage output). Worker tier = sum of all work suitability levels (Kindling + Watering + Planting + Generating + Handiwork + Gathering + Lumbering + Mining + Medicine + Cooling + Transporting + Farming). Overall = Attack + (Worker × 3), weighted toward base utility. Percentiles: S = top 10%, A = 10-30%, B = 30-60%, C = rest. No subjective judgment — purely numbers.
