# Palworld Companion 90天开发路线图

## 项目目标

在90天内完成一个可运行的 Palworld 玩家工具平台 MVP，通过 SEO
长尾关键词获取第一批自然流量，并验证游戏工具站商业模式。

------------------------------------------------------------------------

# 第一阶段：需求验证（第1-30天）

目标：

-   完成核心产品
-   建立SEO基础
-   获取第一批用户

## Week 1：项目基础

任务：

-   确定品牌和域名
-   搭建 Next.js 项目
-   配置数据库
-   建立基础页面结构

页面：

-   首页
-   Tools 页面
-   About 页面
-   Privacy 页面

------------------------------------------------------------------------

## Week 2：开发第一个核心工具

重点：

# Breeding Calculator

功能：

-   输入两个Pal
-   查询可能后代
-   展示结果

SEO页面：

-   palworld breeding calculator
-   palworld breeding chart

------------------------------------------------------------------------

## Week 3：开发第二个工具

# Pal Finder

功能：

根据需求筛选：

-   工作能力
-   属性
-   战斗能力

SEO：

-   palworld pal finder
-   best pal for mining

------------------------------------------------------------------------

## Week 4：开发第三个工具

# Team Builder

功能：

-   Boss队伍推荐
-   PvP组合推荐
-   Farming组合推荐
